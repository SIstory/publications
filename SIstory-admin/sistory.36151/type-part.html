>>

<<

>>

1V sklopu Type se lahko izbere dve vrsti metapodatkovnega zapisa za tip objektov:

  • SIstory Type
  • Type

13.1. SIstory Type

Definicija
Tip gradiva objavljenega na portalu SIstory.
XML element
TYPE
Tip vnosa
Obvezni (Minimalno: 1, Maksimalno: 1)
Uporaba:
Za razliko od sorodnega elementa Type, ki opisuje samo tip gradiva, se s SIstory Type označuje tudi njegovo funkcionalnost na portalu SIstory. Zato je ta tip obvezno potrebno navesti (privzeta vrednost je Besedilo). Pri tem se uporablja kontroliran vokabular:
  • Besedilo
  • Povezava: URL povezava do gradiva, ki ni naložen na SIstory
  • Slika
  • Video
  • Avdio
  • Prezentacija: originalne Microsoft PowerPoint predstavitve
  • Sistem: ni gradivo znanstvene ali kulturne dediščine, temveč pomembno za delovanje sistema poratala SIstory
  • Analogno: za metapodatke brez digitaliziranega gradiva

13.2. Type

Definicija
Dublin Core tip gradiva.
Izvorni element
dc:type
Originalna definicija
"The nature or genre of the resource. Recommended best practice is to use a controlled vocabulary such as the DCMI Type Vocabulary [DCMITYPE]. To describe the file format, physical medium, or dimensions of the resource, use the Format element."
Kontroliran vokabular
Uporablja se kontruliran vokabular dcterms:DCMIType
Originalna definicija
"The set of classes specified by the DCMI Type Vocabulary, used to categorize the nature or genre of the resource.".
XML element
TYPE_DC
Tip vnosa
Obvezni (Minimalno: 0, Maksimalno: neomejeno)
Uporaba
  • Tip gradiva v skladu z Dublin Core kontroliranim vokabularjem:
    • Collection
    • Dataset
    • Event
    • Image
    • InteractiveResource
    • MovingImage
    • PhysicalObject
    • Service
    • Software
    • Sound
    • StillImage
    • Text
  • Ta element je samo dodatna izbira tipa gradiva glede na poglavitni SIstory Type. Z njim dodatno opredelimo tip gradiva.
  • Če so npr. baze podatkov v okviru SIstory Type označene kot Besedilo, se jih pri Type označi kot Dataset. Če je npr. s SIstory Type določeno gradivo označeno kot Povezava, Sistemsko ali Analogno, mu njegov dejanski tip označimo z Dublin Core tipom gradiva.
