33. Za avdiovizualno gradivo

>>

<<

>>

1V sklopu Za avdiovizualno gradivo lahko izberemo sledeča metapodatkovna polja:

  • avLevel
  • avInstantiationType
  • avCarrierType
  • avGauge
  • avOrigDuration
  • avOrigLength
  • avAspectRatio
  • avColourSystem
  • avSoundSystem

33.1. avLevel

Definicija
Stopnja ureditve opisane enote gradiva.
HOPE element
hopeEntity/descriptiveUnit/domain/audioVisualUnit/descriptionLevel
Originalna definicija
"The level of arrangement of the descriptive unit. Domain-specific controlled vocabulary for audio-visual resources is to be specified by HOPE."
XML element
AUDIOVISUALNI[@audiovisElType='avLevel']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: 1)

33.2. avInstantiationType

Definicija
Termin oziroma fraza s katero opišemo tip objekta glede na proces izdelovanja kopij.
HOPE element
hopeEntity/descriptiveUnit/domain/audioVisualUnit/instantiationType
Originalna definicija
"A term or phrase describing the item type relative to the duplication process"
Izvorni element
EN15907:item/instantiation type
Izvorna definicija
"A term or phrase describing the item type relative to the duplication process (e.g. "original negative", "dup negative", "positive", "original positive (reversal film)", "dupe positive", "Lavender", "image negative", "sound negative", "non-film analogue carrier", "non-film digital carrier")."
XML element
AUDIOVISUALNI[@audiovisElType='avInstantiationType']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Uporaba
Suggested controlled vocabulary form EN15907: "original negative", "dupnegative", "positive", "original positive(reversal film)", "dupe positive", "Lavender", "image negative", "sound negative", "non-film analogue carrier", "non-film digital carrier".

33.3. avCarrierType

Definicija
Vrsta fizičnega nosilca, ki se uporablja za manifestacijo avdio-visualnega objekta.
HOPE element
hopeEntity/descriptiveUnit/domain/audioVisualUnit/carrierType
Originalna definicija
"The type of physical carrier used for the manifestation. Digital manifestations may be bound to a physical carrier (such as DVD) or exist without a defined carrier (such as online streaming media, podcasts, etc.)."
Izvorni element
EN15907:format/carrierType
Izvorna definicija
"The type of physical carrier used for the manifestation. Digital manifestations may be bound to a physical carrier (such as DVD) or exist without a defined carrier (such as online streaming media, podcasts, etc.)."
XML element
AUDIOVISUALNI[@audiovisElType='avCarrierType']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)

33.4. avGauge

Definicija
Širina filmskega traku ali kakšnega drugega nosilca.
HOPE element
hopeEntity/descriptiveUnit/domain/audioVisualUnit/gauge
Originalna definicija
"The width of the film stock or other carrier (such as magnetic tape) used for the manifestation. Should include value and unit (e.g. "35 mm", "1/2 in.")"
Izvorni element
EN15907:format/Gauge
XML element
AUDIOVISUALNI[@audiovisElType='avGauge']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Uporaba
Potrebno je navesti mersko enoto, npr. milimetre kot mm.

33.5. avOrigDuration

Definicija
Čas trajanja prve znane filmske manifestacije, merjeno v minutah in sekundah.
HOPE element
hopeEntity/descriptiveUnit/domain/audioVisualUnit/originalDuration
Originalna definicija
"The running time of the first known manifestation of a cinematographic work, measured in minutes and second."
Izvorni element
EN15907:extentUnit=”min.”|”sec.”
XML element
AUDIOVISUALNI[@audiovisElType='avOrigDuration']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)

33.6. avOrigLength

Definicija
Skupni fizični, logični in časovni okvir trajanja manifestacije fimskega dela.
HOPE element
hopeEntity/descriptiveUnit/domain/audioVisualUnit/originalLength
Originalna definicija
"The total physical, logical or temporal extent of the manifestation of a cinematographic work, or of the item from a manifestation. Physical extent is the total length of the medium carrying the manifestation or item. Logical extent is the number of discrete units, e.g. image frames, bytes (octets), etc., of which the manifestation or item is composed. Temporal extent is the running time under normal or specific playback conditions (e.g. a specific frame rate)."
Izvorni element
EN15907:extent unit=”ft.”|”m”
XML element
AUDIOVISUALNI[@audiovisElType='avOrigLength']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)

33.7. avAspectRatio

Definicija
Razmerje med višino in širino slike filma.
HOPE element
hopeEntity/descriptiveUnit/domain/audioVisualUnit/aspectRatio
Originalna definicija
"The ratio between width and height of the image. For anamorphic formats, this element should contain the aspect ratio of the projected image rather than that on the carrier. Some aspect ratios are known not only by numeric values, but also by names (e.g. “full frame”, "Cinemascope")."
Izvorni element
EN15907:format/aspectRatio
XML element
AUDIOVISUALNI[@audiovisElType='avAspectRatio']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)

33.8. avColourSystem

Definicija
Ime barvnega sistema ali procesa.
HOPE element
hopeEntity/descriptiveUnit/domain/audioVisualUnit/colourSystem
Originalna definicija
"A name for the colour system or process."
Izvorni element
EN15907:format/colour/colourSystem
XML element
AUDIOVISUALNI[@audiovisElType='avColourSystem']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)

33.9. avSoundSystem

Definicija
Ime sistema, s katerim je posnet in predvajan zvok.
HOPE element
hopeEntity/descriptiveUnit/domain/audioVisualUnit/soundSystem
Originalna definicija
"The name of the system by which sound is recorded either on the carrier, on a separate medium, or as part of the digital encoding (e.g. Western Electric (Westrex) Movietone, RCA Photophone, Tobis, Dolby Digital, etc.)."
Izvorni element
EN15907:format/soundSystem/systemName
XML element
AUDIOVISUALNI[@audiovisElType='avSoundSystem']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
