4. Creator

>>

<<

>>

Definicija
Naziv organizacije, ki je ustvarila to delo.
Element
dc:creator
Prilagojena originalna definicija
"An entity primarily responsible for making the resource. Examples of a Creator exclude a person and include an organization, or a service."
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Uporaba:
Predviden samo za navajanje organizacij, ki so kot celota navedene kot ustvarjalci dela in pri čemer posamezne osebe teh ustanov niso navedene. Za navajanje osebe je predviden element Avtor, kjer se navaja ime in priimek avtorja.
Atribut
Določi se jezik naziva organizacije. Ista organizacija ima lahko svoj uraden naziv izražen v več jezikih.
