24. Other

>>

<<

>>

1V sklopu Other lahko izberemo sledeča metapodatkovna polja:

  • provenance
  • rightsHolder
  • instructionalMethod
  • accrualMethod
  • accrualPeriodicity
  • accrualPolicy
  • sistoryUnstored

24.1. provenance

Definicija
Navedba kakršne koli spremembe lastništva ali hrambe gradiva, katero je imelo velik vpliv na avtentičnost, integriteto in interpretativno razlago tega gradiva.
Element
dcterms:provenance
Originalna definicija
"A statement of any changes in ownership and custody of the resource since its creation that are significant for its authenticity, integrity, and interpretation. The statement may include a description of any changes successive custodians made to the resource."
XML element
OTHER[@otherType='provenance']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Atribut
Obvezno se določi jezik zapisa.

24.2. rightsHolder

Definicija
Oseba ali organizacija, katera ima ali upravlja avtorsko pravico.
Element
dcterms:rightsHolder
Originalna definicija
"A person or organization owning or managing rights over the resource."
XML element
OTHER[@otherType='rightsHolder']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Uporaba
  • Se obvezno navede v primeru, ko avtor ni več imetnic avtorskih pravic v celoti (temveč v skrajnem primeru samo še imetnik moralnih avtorskih pravic). Tukaj se lahko navede založbo, ki je odkupila avtorske pravice, službo, za katero avtor v okviru opravljanja rednih delovnih obveznosti opravil avtorsko delo, dediči avtorja, ki imajo trenutno v lasti avtorske pravice ipd.
  • Ker to polje nikoli ne bo javno dostopno, se vanj lahko vpisujejo tudi morebiti zapleti glede razčiščevanja avtorskih pravic samega avtorja.
Atribut
Obvezno se določi jezik zapisa.

24.3. instructionalMethod

Definicija
Opis metod spodbujanja učenja, odnosov in spretnosti, ki jih vir posreduje.
Element
dcterms:instructionalMethod
Originalna definicija
"A process, used to engender knowledge, attitudes and skills, that the described resource is designed to support. Instructional Method will typically include ways of presenting instructional materials or conducting instructional activities, patterns of learner-to-learner and learner-to-instructor interactions, and mechanisms by which group and individual levels of learning are measured. Instructional methods include all aspects of the instruction and learning processes from planning and implementation through evaluation and feedback."
XML element
OTHER[@otherType='instructionalMethod']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Uporaba
Trenutno se ta podatek ne navaja. Začel se bo navajati, če bomo v okviru diferencijacije objavljenega gradiva glede na občinstvo (audience) hoteli dodati še informacije o metodah, ki se jih uporablja pri posredovanju tega gradiva.
Atribut
Obvezno se določi jezik zapisa.

24.4. accrualMethod

Definicija
Metoda, po kateri se gradivo dodaja zbirki.
Element
dcterms:accrualMethod
Originalna definicija
"The method by which items are added to a collection."
XML element
OTHER[@otherType='accrualMethod']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Atribut
Obvezno se določi jezik zapisa.

24.5. accrualPeriodicity

Definicija
Frekvenca s katero se gradivo dodaja zbirki.
Element
dcterms:accrualPeriodicity
Originalna definicija
"The frequency with which items are added to a collection."
XML element
OTHER[@otherType='accrualPeriodicity']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Atribut
Obvezno se določi jezik zapisa.

24.6. accrualPolicy

Definicija
Politika po kateri se gradivo dodaja k zbirki.
Element
dcterms:accrualPolicy
Originalna definicija
"The policy governing the addition of items to a collection."
XML element
OTHER[@otherType='accrualPolicy']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Atribut
Obvezno se določi jezik zapisa.

24.7. SIstory Unstored

Definicija
Polje za shranjevanje metapodatkov, ki jih zaradi vsebine ni mogoče shraniti v nobenemu drugemu metapodatkovnemu polju.
XML element
OTHER[@otherType='Sistory Unstored']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Uporaba
Uporablja se samo v primeru, ko določenih metapodatkov ni mogoče shraniti kjer koli drugje.
Atribut
Obvezno se določi jezik zapisa.
