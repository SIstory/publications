31. Knjižničarski elementi

>>

<<

>>

1V sklopu Knjižničarski elementi lahko izberemo sledeča metapodatkovna polja:

  • libLevel
  • libGenMaterialDesignation
  • libSpecMaterialDesignation
  • libExtent
  • libPubFrequency
  • libDimensions
  • libOtherPhysDetails

31.1. libLevel

Definicija
Stopnja ureditve opisane enote gradiva.
HOPE element
hopeEntity/descriptiveUnit/domain/libraryUnit/descriptionLevel
Originalna definicija
"The level of arrangement of the descriptive unit. Suggestion: series, periodicals, monographs, issues."
XML element
KNJIZNICARSKI[@knjizniElType='libLevel']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: 1)
Uporaba
  • Npr. zbirka, monografija, periodična publikacija, znanstvena revija, članek, poglavje, znanstveni članek, strokovni članek, recenzija, poročilo ...
  • Priporočljiv je kontroliran vokabular.

31.2. libGenMaterialDesignation

Definicija
Značilnosti in splošni podatki o tipu gradiva, ki se nanaša na vsebino publikacije.
HOPE element
hopeEntity/descriptiveUnit/domain/libraryUnit/generalMaterialDesignation
Originalna definicija
"The characteristics and general material type of content of the resource"
Izvorni element
marc:245$k
Izvorna definicija
"$k - Form: Term that is descriptive of the form of the described materials, determined by an examination of their physical character, subject of their intellectual content, or the order of information within them (e.g., daybooks, diaries, directories, journals, memoranda, etc.)."
XML element
KNJIZNICARSKI[@knjizniElType='libGenMaterialDesignation']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Uporaba
Priporočljiva je uporaba kontroliranega vokabularja, npr. dnevnik, časopis, memorandum, stenografski zapisniki, statistični viri ipd.

31.3. libSpecMaterialDesignation

Definicija
Opis fizičnega izgleda publikacije.
HOPE element
hopeEntity/descriptiveUnit/domain/libraryUnit/specificMaterialDesignation
Originalna definicija
"Physical description of the described item, including its extent, dimensions, and such other physical details as a description of any accompanying materials and unit type and size."
Izvorni element
marc:300
Izvorna definicija
"Physical description of the described item, including its extent, dimensions, and such other physical details as a description of any accompanying materials and unit type and size."
XML element
KNJIZNICARSKI[@knjizniElType='libSpecMaterialDesignation']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Uporaba
  • Splošen opis izgleda publikacije. Obseg publikacije se vpisuje v polje libExtent, dimenzije publikacije se vpisuje v polje libDimensions, fizične značilnosti pa v polje libOtherPhysDetails.
  • Poleg splošnih stavčnih opisov izgleda publikacije, se v tem polju bolj natančno opisuje še priloženi material (npr. priloženi zemljevidi).

31.4. libExtent

Definicija
Obseg publikacije.
HOPE element
hopeEntity/descriptiveUnit/domain/libraryUnit/extent
Originalna definicija
"Number of pages, volumes, etc. of the physical or digital unit or units constituting the resource, adding other measures of extent as appropriate."
Izvorni element
marc:300$a
Izvorna definicija
"Number of physical pages, volumes, cassettes, total playing time, etc., of of each type of unit."
XML element
KNJIZNICARSKI[@knjizniElType='libExtent']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Uporaba
Število strani (analogne) publikacije, število knjig določenega dela, število volumnov ipd.

31.5. libPubFrequency

Definicija
Frekvenca izhajanja publikacije.
HOPE element
hopeEntity/descriptiveUnit/domain/libraryUnit/publicationFrequency
Originalna definicija
"Complete statement, exclusive of dates, of the current publication frequency"
Izvorni element
marc:310$a
Izvorna definicija
"Current stated publication frequency of either an item or an update to an item. $a - Current publication frequency: Complete statement, exclusive of dates, of the current publication frequency."
XML element
KNJIZNICARSKI[@knjizniElType='libPubFrequency']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)

31.6. libDimensions

Definicija
Dimenzije publikacije.
HOPE element
hopeEntity/descriptiveUnit/domain/libraryUnit/dimensions
Originalna definicija
"Dimensions of the resource, such as size and duration."
Izvorni element
marc:300$c
Izvorna definicija
"Expressed in centimeters, millimeters, or inches; may include a parenthetical qualifier giving the format of the item (e.g., (fol.), (8vo)). "
XML element
KNJIZNICARSKI[@knjizniElType='libDimensions']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Uporaba
Dimenzije vira, npr. velikost (v centimetrih), čas trajanja posnetka ipd.

31.7. libOtherPhysDetails

Definicija
Ostale fizične značilnosti publikacije.
HOPE element
hopeEntity/descriptiveUnit/domain/libraryUnit/otherPhysicalDetails
Originalna definicija
"Other physical characteristics of the resource, such as method of production, colour, material from which the resource is made."
Izvorni element
marc:300$b
Izvorna definicija
"Physical characteristics such as illustrative matter, coloration, playing speed, groove characteristics, presence and kind of sound, number of channels, motion picture presentation format, etc."
XML element
KNJIZNICARSKI[@knjizniElType='libOtherPhysDetails']
Tip vnosa
Opcijski (Minimalno: 0, Maksimalno: neomejeno)
Uporaba
Barve, material, iz katerega je publikacija izdelana, način izdelave.
